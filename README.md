# Ehcoo
一个 `ehco` 配置便捷脚本

## 优点
- [x] 简单配置 `ehco` 隧道
- [x] 支持开机自启
- [x] 支持中转机器 
- [x] 支持落地机器

## 更新
- 支持添加多个中转
- 支持国内机器使用

## 计划功能
- 判断端口是否被占用
- 流量统计(暂时可能不会实现)

## 兼容性
支持 `CentOS` `Ubuntu` `Debian` 等 `Linux` 系统的所有版本

## 使用
```shell
wget -O ehco.sh https://cdn.jsdelivr.net/gh/missuo/Ehcoo/ehco.sh && bash ehco.sh
```
